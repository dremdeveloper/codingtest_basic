# 코딩 테스트 성능 비교 예시 (JavaScript)

Python 성능 비교 예제를 JavaScript로 옮긴 버전입니다.

## 파일 목록
- `membership_lookup.js`: `Array.includes` vs `Set.has`
- `string_building.js`: 문자열 `+=` 반복 vs `Array.join`
- `range_sum_query.js`: 매 질의 합산 vs 누적합(prefix sum)
- `deduplication.js`: 배열 기반 중복 제거 vs `Set` 기반 중복 제거

## 실행 예시
```bash
node javascript/performances/membership_lookup.js
node javascript/performances/string_building.js
node javascript/performances/range_sum_query.js
node javascript/performances/deduplication.js
```
