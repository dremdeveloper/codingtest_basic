# 코딩 테스트 성능 비교 예시 (C++)

`python/performances`의 성능 비교 예시를 C++로 옮긴 폴더입니다.

## 파일 목록

- `membership_lookup.cpp` : `vector` 선형 탐색 vs `unordered_set` 조회
- `string_building.cpp` : 문자열 `+=` 반복 vs 미리 `reserve` 후 append
- `range_sum_query.cpp` : 구간합 매번 계산 vs prefix sum
- `deduplication.cpp` : 결과 벡터 선형 검사 vs `unordered_set` 보조
