# 코딩 테스트 대비 기본 알고리즘 예시 (JavaScript)

Python 예제를 기준으로 동일한 구성의 JavaScript 버전을 제공합니다.

## 파일 목록

### 탐색 / 그래프
- `binary_search_recursive.js`: 이진 탐색 (재귀)
- `binary_search_iterative.js`: 이진 탐색 (반복)
- `dfs_recursive.js`: DFS (재귀)
- `dfs_iterative.js`: DFS (반복, 스택)
- `bfs_queue.js`: BFS (큐)
- `dijkstra_heap.js`: 다익스트라 최단 경로 (우선순위 큐)
- `topological_sort_kahn.js`: 위상 정렬 (Kahn)
- `union_find.js`: 유니온 파인드

### 정렬
- `merge_sort_recursive.js`: 병합 정렬 (재귀)
- `merge_sort_iterative.js`: 병합 정렬 (반복, Bottom-Up)
- `quicksort_recursive.js`: 퀵 정렬 (재귀)
- `quicksort_iterative.js`: 퀵 정렬 (반복, 스택)

### 수학 / 분할정복
- `factorial_recursive.js`: 팩토리얼 (재귀)
- `factorial_iterative.js`: 팩토리얼 (반복)
- `fibonacci_recursive.js`: 피보나치 수열 (재귀)
- `fibonacci_iterative.js`: 피보나치 수열 (반복)
- `gcd_recursive.js`: 최대공약수 (재귀)
- `gcd_iterative.js`: 최대공약수 (반복)
- `power_recursive.js`: 거듭제곱 (재귀, 분할 정복)
- `power_iterative.js`: 거듭제곱 (반복, 이진 지수법)

### 동적 계획법
- `knapsack_01_dp.js`: 0/1 배낭 문제
- `lis_dp_binary_search.js`: 최장 증가 부분 수열 (LIS, O(n log n))

총 22개 예시 파일을 포함합니다.
